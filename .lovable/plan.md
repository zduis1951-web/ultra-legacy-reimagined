
# تحسين صفحة الخدمات (Services) بمستوى فخامة عالمي

## الرؤية
اعادة تصميم صفحة الخدمات بالكامل لتتطابق مع مستوى الفخامة الموجود في الصفحة الرئيسية (WhyChooseUs, AboutSection) مع نفس الانماط: خطوط ذهبية زخرفية، ايقونات داخل اطارات ماسية، تاثيرات hover متقدمة، ومسافات تنفس كبيرة.

---

## التغييرات على ملف `src/pages/Services.tsx`

### 1. قسم Hero العلوي للصفحة
- اضافة خلفية متدرجة فاخرة مع خطوط ذهبية علوية وسفلية
- توحيد نمط العنوان مع باقي الموقع (خط ذهبي + تاج + عنوان رئيسي + خط ذهبي)
- زيادة المسافات العمودية الى `py-32 md:py-44`

### 2. بطاقات الخدمات الاربع
- تحويل الايقونات الى نمط الاطار المزدوج (double border) كما في WhyChooseUs
- اضافة خط ذهبي متحرك اعلى واسفل كل بطاقة عند hover (يتمدد من المنتصف)
- زيادة padding الداخلي الى `p-10`
- اضافة خط فاصل ذهبي صغير بين العنوان والوصف (`w-8 h-px bg-gold/40`)
- تحسين الظلال عند hover الى `shadow-[0_25px_80px_-20px_rgba(0,0,0,0.12)]`
- تحسين تاثير الحدود عند hover الى `hover:border-gold/40`

### 3. قسم "لماذا تختارنا"
- اضافة خلفية متدرجة فاخرة بدلا من `bg-muted/30` البسيطة
- تحويل الايقونات الى نمط الاطار الماسي المدور (rotate-45) كما في WhyChooseUs
- اضافة خطوط ذهبية زخرفية علوية وسفلية للقسم
- اضافة خط فاصل ذهبي صغير تحت كل عنوان
- زيادة padding البطاقات الى `p-10` مع خلفية `bg-card` وحدود

### 4. قسم CTA جديد اسفل الصفحة
- اضافة قسم دعوة للتواصل اسفل الصفحة بخلفية داكنة (`bg-foreground text-background`)
- زر "تواصل معنا" بلون ذهبي
- خطوط ذهبية زخرفية

---

## التفاصيل التقنية

### الملف المتاثر:
- `src/pages/Services.tsx` (اعادة كتابة شاملة)

### الانماط المستخدمة (مطابقة للصفحة الرئيسية):
- ايقونات ماسية: `rotate-45` مع `border` مزدوج
- خطوط ذهبية متحركة: `w-0 group-hover:w-full transition-all duration-700`
- فواصل ذهبية: `h-px bg-gold mx-auto`
- ظلال عميقة: `shadow-[0_25px_80px_-20px_rgba(0,0,0,0.12)]`
- مسافات موحدة: `py-32 md:py-44`
- تتبع الحروف: `tracking-[0.4em]`
