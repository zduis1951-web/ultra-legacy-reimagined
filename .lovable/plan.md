

# تحسينات فائقة الجودة - بيت الشموخ

## المشاكل الحالية المكتشفة

1. **صورة الهيرو لا تظهر** - الخلفية رمادية فارغة بدون صورة عقارية
2. **نصوص النافبار** - تظهر على خلفية شفافة مع تباين ضعيف، والشعار غير واضح
3. **أزرار الهيرو والنص الفرعي غير مرئية** - لون النص يندمج مع الخلفية الرمادية
4. **اللغة ليست ديناميكية بالكامل** - بعض النصوص hardcoded بالإنجليزية (اسم الشركة في النافبار والفوتر، الأرقام، أسماء الشبكات الاجتماعية)
5. **لا توجد صور برج خليفة أو صور فاخرة حقيقية**

---

## خطة التنفيذ

### 1. إصلاح صورة الهيرو + صور احترافية
- استبدال hero-bg.jpg بصورة luxury عالية الجودة من Unsplash (برج خليفة / ناطحة سحاب فاخرة)
- استخدام صور عقارية حقيقية بدقة عالية لجميع الأقسام (property1, property2, property3, about-bg)
- إضافة gradient overlay أقوى وأوضح على الهيرو لضمان وضوح النص

### 2. إصلاح النافبار بالكامل
- تحسين تباين النص على الخلفية الشفافة (نص أبيض على hero، أسود بعد scroll)
- جعل اسم الشركة "بيت الشموخ" و "عقارات" ديناميكي بالعربي/إنجليزي
- إضافة الشعار بشكل واضح
- تحسين زر تبديل اللغة بتصميم أنيق

### 3. جعل اللغة ديناميكية 100%
- إضافة ترجمات لاسم الشركة (Bayt Alshumukh / بيت الشموخ)
- إضافة ترجمات لـ "Real Estate" / "عقارات"
- ترجمة أسماء الشبكات الاجتماعية في الفوتر
- ترجمة نص حقوق النشر
- ضمان كل نص في الموقع يستخدم نظام t() للترجمة

### 4. تحسينات التصميم الفاخر
- تحسين Hero Section: gradient أقوى، أزرار واضحة، نص فرعي مقروء
- تحسين ألوان النافبار: أبيض على الهيرو، أسود بعد التمرير
- إضافة particle/shimmer effects خفيفة على الذهبي
- تحسين hover effects على جميع البطاقات

### 5. تحديث جميع الملفات

## التفاصيل التقنية

### LanguageContext - مفاتيح جديدة
```
"brand.name" -> "Bayt Alshumukh" / "بيت الشموخ"
"brand.subtitle" -> "Real Estate" / "عقارات"
"social.instagram" -> "Instagram" / "انستغرام"
"social.facebook" -> "Facebook" / "فيسبوك"
"social.linkedin" -> "LinkedIn" / "لينكدإن"
```

### Navbar - إصلاح التباين
- حالة غير مسحوبة (على الهيرو): `text-white` لجميع الروابط
- حالة مسحوبة (بعد scroll): `text-foreground` مع خلفية بيضاء
- اللغة الحالية النشطة: `text-gold` دائما

### Hero - إصلاح الظهور
- استخدام صور Unsplash بروابط مباشرة كـ fallback إذا لم تعمل الملفات المحلية
- gradient overlay: `bg-gradient-to-r from-black/80 via-black/50 to-black/20`
- نص أبيض واضح مع ظل خفيف
